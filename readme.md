Ministream is a lightweight full-stack video streaming platform inspired by YouTube.

It allows users to upload videos, watch them directly in the browser, like/unlike content, and interact through comments. The project is built to demonstrate modern full-stack development using FastAPI for the backend and React (Vite + TypeScript + Tailwind CSS) for the frontend.

Features:

- User authentication using token-based login
- Video upload and streaming
- Like / unlike functionality
- Comment system
- Video listing and watch page
- RESTful API built with FastAPI
- Modern React frontend with Vite and Tailwind CSS

Tech Stack:
Backend: FastAPI, Python  
Frontend: React, TypeScript, Tailwind CSS  
Database: SQLite
